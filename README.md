# Description

The Long Way Home

A quiet, atmospheric walking simulator about memory, routine… and what happens when those memories begin to slip away.

Walk the same path home each day, through sunlit streets, quiet gardens, and familiar places.
At first, everything feels warm. Comforting.

But the world doesn’t stay the same.

Details begin to change.
Sounds feel… wrong.
Some memories become harder to reach.

And some things refuse to be forgotten.

# Features
- Hand-painted Pixel Art
- A nostalgic world that slowly shifts and distorts over time.
- A Story Told Through Environment
- No dialogue-heavy exposition — everything is revealed through what you see, hear, and lose.
- Evolving World
- The same journey home, replayed across different times, weather… and states of mind.
- Psychological Shift
- What begins as peaceful reflection gradually becomes something else.

# Controls
- A / D / ← / → — Move
- W / ↑ — Jump (some levels)
- A — Interact

# Instructions

Each day, you return home from work.

Follow the same route through familiar streets and spaces.
Interact with objects and revisit places.

Walk to your house at the end of the road and press A to enter your house.

Some paths open.
Some details fade.
Some memories don’t stay the same.

Keep moving forward.
Go home.

# Optimisation

This game is designed to run smoothly on low-end hardware.

- Persistent Levels
Instead of rebuilding each level every loop, the world is continuously modified.
This reduces processing load and also reinforces the feeling of gradual change.
- Layered Animation System
Moving elements are rendered as overlays on top of the background, rather than being part of it.
This prevents background redraw slowdowns and keeps animations smooth even on weaker systems.
